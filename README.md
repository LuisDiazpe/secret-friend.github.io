# Amigo Secreto 🎁

Este es un proyecto web para jugar al **Amigo Secreto** de forma sencilla y divertida. Cada jugador ingresa su nombre y descubre a quién le tocó como amigo secreto, evitando duplicaciones y asegurando que nadie se asigne a sí mismo.

## Características
- Los jugadores se registran automáticamente desde una base de datos local (`participants.json`).
- Cada jugador debe validar su nombre antes de descubrir a su amigo secreto.
- Una vez asignado un amigo secreto, el jugador no puede volver a participar.
- Los nombres se eliminan dinámicamente para evitar duplicaciones y garantizar un juego justo.
